# ProjectJangu

## Development server

To start a local server, run:

```bash
ng serve
```

To add your TMDB API key, create (or open) the file at:

```bash
src/environments/environment.ts
```

and inside it define your key, for example:

```ts
// src/environments/environment.ts
export const api_key = 'YOUR_TMDB_API_KEY';
```
